Advanced — manual elements & CORS proxy

AMSAT's bulletin server doesn't send CORS headers, so a small proxy is used to fetch it from the browser. Leave blank to use the default chain.

Sheets are the same vector output as OrbitDeck's OSCARLOCATOR export, generated in-browser — nothing is uploaded. Only AMSAT GP elements (and, if you ask, your location) are fetched. Coastlines are Natural Earth 110m. Print at 100% / actual size so the base map and transparency overlays register.
